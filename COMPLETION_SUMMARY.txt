## 🎯 The Cascade Works Like This

```
User selects Component (e.g., "STAR")
         ↓
System shows Projects: [STAR, STAR PNGO, STAR UPG]
         ↓
User selects Project (e.g., "STAR")
         ↓
System shows Zonal Areas: [Noakhali, Barisal, Rangpur]
         ↓
User selects Zonal Area (e.g., "Barisal")
         ↓
System shows DM Areas: [Patuakhali, Barisal sadar, Bagerhat, Jessore]
         ↓
User selects DM Area (e.g., "Patuakhali")
         ↓
System shows Branches: [STAR Patuakhali Sadar, STAR Bauphal, ...]
         ↓
User selects Branch (optional)
         ↓
System shows Districts: [...] (optional)
         ↓
Form Data Complete & Ready to Submit
```
